# App Guide

SupoClip opens directly on the clip creation workflow for connected local users.

## Main Flow

1. Sign in or create a local account.
2. Paste a YouTube URL or upload a video.
3. Pick caption style, output format, subtitles, and font options.
4. Start processing.
5. Watch progress and open generated clips as they arrive.

## Task History

The header links to all generations. Each task stores:

- Source title and type
- Processing status and progress
- Clip count
- Generated clips and edit metadata

## Clip Editing

On a task page, clips can be reviewed, trimmed, split, merged, and exported. Caption text can also be adjusted after generation.

## Settings

The settings page stores default caption style:

- Font family
- Font size
- Font color

These defaults are tied to the local account, so several profiles can keep separate preferences on the same machine.
